# 测试版 V0.8.0 版本说明

**版本号：** 0.8.0  
**发布日期：** 2026-06-30

## 更新内容

### 安装版（首次安装）

- 新增 **Inno Setup 安装程序** `LuxeLead_Setup_V0.8.0.exe`
- 双击安装，自动创建开始菜单/桌面快捷方式，无需手动解压 zip

### 在线更新加速

- 支持 **增量更新包**：小版本只下载/覆盖变更文件，显著缩短下载与安装时间
- 完整 zip 仍保留，供首次安装或跨版本升级使用

### 历史修复（V0.7.9）

- PowerShell 独立进度窗，避免更新时文件占用
- robocopy 安装，`LuxeLead/` 顶层目录结构

## 使用说明

| 场景 | 方式 |
|------|------|
| 首次安装 | 下载 `LuxeLead_Setup_V0.8.0.exe` 或解压完整 zip |
| 在线更新 | 帮助 → 检查更新（V0.7.9 起优先下载增量包） |
| 手动目录 | 进入 `LuxeLead` 文件夹，双击 exe |
